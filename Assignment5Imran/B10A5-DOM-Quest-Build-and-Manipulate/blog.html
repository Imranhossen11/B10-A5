Important F A Q

What is the document object Model (DOM)?

Answer: The Document Object Model (DOM) is a programming interface that represents a web page's structure and allows programs to change its content, style, and structure.

How do you Select an element from the DOM?

Answer: To select an element from the Document Object Model (DOM) in JavaScript, you use methods like getElementById to access an element by its unique ID, getElementsByClassName to get elements with a specific class name, or querySelector to select an element using a CSS selector that can combine ID, class, tag name, and attributes for more complex selections; each method returns the selected element(s) as a DOM node object.

What is event delegation in the context of the DOM,and why is it useful?

Answer: In the context of the DOM (Document Object Model), "event delegation" is a technique where you attach a single event listener to a parent element, instead of attaching listeners to multiple child elements, and then use the event bubbling mechanism to identify which child element triggered the event, making it a powerful way to handle events for dynamically added or numerous child elements, improving performance and code maintainability

How do you manipulate an elements attributes and styles and styles using the DOM?

Answer: To manipulate an element's attributes and styles using the DOM in JavaScript, you first need to select the desired element using methods like getElementById, querySelector, or getElementsByClassName, then use the setAttribute method to change attributes and the style property to modify CSS styles directly on the element.
